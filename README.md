# Beschreibung

Ein sehr einfacher Timer, der für das Format Sprechen & Zuhören verwendet werden kann.
Die Kleingruppen können ihr jeweiliges Setting einstellen (Anzahl der Beteiligten, Dauer und Runden). 
Der Timer ist so gestaltet, dass er sich gut auf dem Smartphone öffnen und dann in die Mitte des Gesprächstischs legen lässt.
